# Development Setup

## Clone the Repository

```
git clone https://github.com/ZYRACX/rcs
```

## Install Dependencies

Server:

```
cd server
npm install
```

Client:

```
cd client
npm install
```

## Run Development Servers

Server:

```
npm run dev
```

Client:

```
npm start
```

## Environment Variables

Create a `.env` file.

Example:

```
APPWRITE_ENDPOINT=
APPWRITE_PROJECT_ID=
APPWRITE_API_KEY=
PORT=8000
```
