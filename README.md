# RCS Project

## Overview

This repository contains the source code for the RCS project, a multi-component application comprising a web client, a Node.js backend server, and a Discord bot. The project leverages modern web technologies and a robust backend to deliver interactive features and services.

## Features

### Web Client
- User authentication (Login, Register)
- Admin functionalities (Inventory Inspector, Overview, Item Management, Player Management)
- Game-related pages (Overview, Inventory)

### Backend Server
- RESTful API endpoints for authentication, economy, player information, inventory, mining, and administrative tasks.
- Integration with Appwrite for backend services.

### Discord Bot
- Discord commands for game interaction (e.g., balance, inventory, market, mine, buy, quicksell, search, register, help).
- Connects to a MySQL database.

## Technologies Used

### Web Client
- **Frontend Framework**: React
- **Build Tool**: Vite
- **Styling**: TailwindCSS

### Backend Server
- **Runtime**: Node.js
- **Web Framework**: Express.js

### Discord Bot
- **Runtime**: Node.js
- **Discord API Wrapper**: discord.js
- **Database**: MySQL2
- **Environment Variables**: Dotenv
- **Development**: Nodemon

## Setup and Installation

To set up the RCS project locally, follow these steps:

### Prerequisites
- Node.js (LTS version recommended)
- npm or Yarn
- MySQL Server
- Appwrite Instance (with configured project and collections)
- Discord Bot Token and Application ID

### 1. Clone the Repository

```bash
git clone https://github.com/ZYRACX/rcs.git
cd rcs
```

### 2. Backend Server Setup

Navigate to the `server` directory, install dependencies, and create a `.env` file.

```bash
cd server
npm install
```

Create a `.env` file in the `server` directory with the following variables:

```
PORT=8000
APPWRITE_ENDPOINT=YOUR_APPWRITE_ENDPOINT
APPWRITE_PROJECT_ID=YOUR_APPWRITE_PROJECT_ID
APPWRITE_API_KEY=YOUR_APPWRITE_API_KEY
```

Start the server:

```bash
npm run dev
```

### 3. Web Client Setup

Navigate to the `client` directory, install dependencies, and create a `.env` file.

```bash
cd ../client
npm install
```

Create a `.env` file in the `client` directory with the following variables:

```
VITE_APPWRITE_ENDPOINT=YOUR_APPWRITE_ENDPOINT
VITE_APPWRITE_PROJECT_ID=YOUR_APPWRITE_PROJECT_ID
```

Start the client development server:

```bash
npm run dev
```

### 4. Discord Bot Setup

Navigate to the `discord` directory, install dependencies, and configure the database connection.

```bash
cd ../discord
npm install
```

**Database Configuration:**

Edit `connection.js` to configure your MySQL database credentials. By default, it's set to `localhost`, `root` user with no password, and `rcs` database.

```javascript
const db = mysql.createConnection({
    host: 'localhost',
    user: 'root',
    password: '', // Your MySQL root password
    database: 'rcs'
});
```

Create a `.env` file in the `discord` directory with your Discord bot token:

```
DISCORD_BOT_TOKEN=YOUR_DISCORD_BOT_TOKEN
```

Start the Discord bot:

```bash
npm run dev
```

## Usage

- Access the web client through your browser (usually `http://localhost:5173`).
- Interact with the Discord bot using the configured prefix (e.g., `rc` followed by commands like `rc help`).
